## CNN Model Results

### 📷 Qualitative Results
The qualitative results show the fused images generated using the CNN model.  
The model captures basic spatial features but lacks fine detail preservation.

### 📊 Quantitative Results
The quantitative evaluation includes metrics such as **PSNR, SSIM, Mutual Information (MI), RMSE, and Entropy**.  
The CNN model shows moderate performance, with lower SSIM and PSNR values and higher RMSE compared to the proposed model.

### ✅ Conclusion
CNN provides a baseline for image fusion but does not effectively capture global dependencies.
